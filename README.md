# Life Farm

This is a life care service for who wants to schedule and manage their daily routines.

## Code Name

whistle

- 이무진-휘파람 노래가 나와서 그냥 이걸로 결정함

## Language

Rust

## Setup (Mac)

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
rustc --version
cargo new whistle
cargo build
cargo run
